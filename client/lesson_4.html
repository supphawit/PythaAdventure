# Write your code
วิธีเล่น คำใบ้ 1 คำใบ้ 2 คำใบ้ 3

How to play

การประกาศ Function มีรูปแบบดังภาพ
โดยจะมีรูปแบบคือ def ชื่อFunction(): ดังภาพ
*อย่าลืมใส่เครื่องหมาย Colon*
และทุก Function จะต้องมีการ Return อะไรซักอย่างออกมา
โดยในตัวอย่างจะเป็นการ Return คำว่า hello ออกมา
Function สามารถเรียกใช้ได้โดยการพิมพ์ชื่อ Function
ตามด้วยเครื่องหมายวงเล็บเปิดและปิดดังภาพ

How to play

เราสามารถประกาศ Function ที่สามารถรับค่ามาประมวลผล
แล้วค่อย Return โดยมีรูปแบบคือ def ชื่อฟังก์ชั่น(ตัวแปรที่1,ตัวแปรที่ 2) ดังภาพ
ภายในการ Return เราสามารถระบุได้ว่าตัวแปรที่รับมาทั้ง 2 ตัวจะนำมาทำอะไรต่อ
ตัวอย่างในภาพคือการนำมาบวกกัน
การเรียกใช้ Function ที่สามารถรับค่ามาประมวลผลสามารถทำได้โดย
พิมพ์ชื่อฟังก์ชั่นแล้วส่งตัวแปรเข้าไปตามที่ประกาศไว้ ดังภาพ
