เงื่อนไข if elif else

เงื่อนไข if elif else มีวิธีใช้คือเป็นคำสั่งในการแยก condition โดยแปลง่ายๆ ว่าเป็นคำว่า "ถ้า" "แล้ว"
โดยในหมายเลข 1 คือเป็นสิ่งที่ต้องมีในเงื่อนไข
หมายเลข 2 จะใช้ในกรณีที่มีมากกว่า 2 เงื่อนไข และเงื่อนไขสุดท้ายจะปิดด้วยหมายเลข 3
หากต้องการให้เงื่อนไขแรกไปทางซ้าย ให้ Print คำว่า Left
หากต้องการให้เงื่อนไขที่สองไปทางขวา ให้ Print คำว่า Right
