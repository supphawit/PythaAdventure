เรียนรู้ก่อนจะเล่น

print

บทเรียนนี้จะเป็นการสอนเกี่ยวการใช้คำสั่ง Print เพื่อแสดงข้อความออกมาทางหน้าจอ ซึ่งเป็นพื้นฐานในการเรียนภาษาทางคอมพิวเตอร์ใหม่ทุกภาษา

if else

บทเรียนนี้จะเป็นการสอนเกี่ยวกับการใช้คำสั่ง IF ELSE ในการทำเงื่อนไขต่างๆ ในภาษา Python ผู้เรียนจะได้ทราบถึงวิธีการใช้เงื่อนไข IF ELSE และรูปแบบต่างๆ ของเงื่อนไข

Learn Unavailable
for loop

บทเรียนนี้จะเป็นการสอนเกี่ยวการใช้ For Loop เพื่อแสดงข้อความออกมาทางหน้าจอ โดยจะมีการสอนพร้อมยกตัวอย่างประกอบในฉาก

Learn Unavailable
function

บทเรียนนี้จะเป็นการสอนเกี่ยวการสร้าง Function โดยจะมีการสอนการเรียกฟังก์ชั่นมาใช้พร้อมยกตัวอย่างในฉาก

List

บทเรียนนี้จะเป็นการสอนเกี่ยวกับการประกาศและสร้าง List รวมถึงเมธอดพื้นฐานและการใช้ For loop แสดงค่าออกมา
